Oddbits

An odd collection of useful web tools

Oddbits is open source and free. Our goal is to build useful tools you can run in the browser, use from the CLI, or plug into your own projects one bit at a time.

How to Use in Your Project

Install

npm install @oddbits/<bit-name>

Use

import { someUtility } from '@oddbits/<bit-name>';
const result = await someUtility(input, options);
npx @oddbits/<bit-name> --help

Use Source and How To in each bit window for exact options.
