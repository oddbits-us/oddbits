Oddbits

A growing collection of rad developer tools and utilities

Oddbits is an open-source project dedicated to bringing back the fun and flavor of classic 16-bit software into modern, powerful web utilities. Whether you need to process images, generate code, or tweak assets, we've got the tools for your cyberpunk toolbelt.

Available Tools

  • 🖼️

    ImageBits

    Resize, optimize, and convert images to webp, avif, png, jpg

    View Source →

How to Use in Your Project

Install via NPM

npm install @oddbits/imagebits

Integrate our fast, optimized tools directly into your NodeJS or frontend builds.

Basic Usage

import { processImage } from '@oddbits/imagebits';

// Convert any image to WebP with max 800px width
const result = await processImage(file, {
  format: 'webp',
  maxDimension: 800,
  quality: 0.92
});
