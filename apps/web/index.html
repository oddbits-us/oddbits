📝Readme.md

Readme

Oddbits is an open source collection of small tools (bits) you can use three ways:

  • In the browser — try any bit right here on this desktop.
  • From the CLI — npx @oddbits/<bit> --help
  • In your code — npm install @oddbits/<bit>

Same API surface across browser and Node — bundlers pick the right build automatically.

Each bit window has a More info button with a quick example. For full docs, install instructions, and source, head to GitHub.
